# PAIN

A browser-based dungeon maze game inspired by Christopher Manson's *Maze*, Myst, and classic Zelda dungeons.

## Play

Open `index.html` in any modern browser. No install, no dependencies, no build step.

## About

A kid is trapped in a dungeon. Find the master key. Escape.

- **50 rooms** across 7 visual themes — stone halls, libraries, courtyards, dungeons, hallways, living rooms, crawlspaces
- **3/4 top-down view** with an SNES-style pixel art character
- **Arrow key / WASD movement** with walk animation
- **Numbered doors** with brass plaques — each leads to another room
- **Room 24 is death** — fall into The Pit, game over
- **Master key** hidden deep in the maze — unlocks the way home
- **Secret puzzle room** with a hidden teleport
- **Procedural audio** — ambient drone, Zelda-style dungeon melody, sound effects — all generated in code, zero audio files
- **Intro cutscene** and **win sequence**

Built from a hand-drawn maze on a stack of paper.

## Tech

Single HTML file. Canvas rendering, Web Audio API, vanilla JavaScript. No frameworks, no assets, no server. Deployable as an Azure Static Web App.

## Controls

| Key | Action |
|-----|--------|
| Arrow keys / WASD | Move |
| R | Restart (after death or win) |
